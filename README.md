# Duo Liu's Academic Homepage

This repository hosts the static personal homepage for [Duo Liu](https://cat-blizzard.github.io/).

The design is inspired by the structure of [Academic Pages](https://github.com/academicpages/academicpages.github.io): an author profile, academic navigation, research interests, news, education, publications, awards, projects, and service sections.

## Local Preview

Open `index.html` directly in a browser, or run a small static server from the repository root:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
